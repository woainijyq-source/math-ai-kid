# 非数学主题 Progression 路线图

## 这份文档解决什么问题

当前项目已经把 daily 主题扩成 5 条线：

1. 数学思维
2. 观察规律
3. 为什么与解释
4. 公平与选择
5. 假设与预测

其中数学思维已经有比较清晰的 progression 文档。剩下 4 条主题虽然已经有：

- theme playbook
- child signal
- 会话内动态适应
- 跨会话轻量升级

但如果不把每条线的 level 标准写清楚，就容易出现两个问题：

1. 系统已经能“升级”，但不知道在教育上到底升了什么
2. 后续补种子时，没有统一的分层依据

这份文档就是当前项目里 **非数学主题** 的 progression 路线图。

## 总体结构

非数学 4 条主题都采用统一的 3 层模型：

- **L1：进入与看见**
- **L2：说清与比较**
- **L3：换条件与迁移**

这和数学思维不同：

- 数学思维目前更适合用 4 段 progression
- 非数学 4 条线先用轻量的 3 层模型就够

原因：

- 这些主题目前更像 daily 陪聊主线
- 不适合一开始就做得过重

## 谁来主导

### 1. 你来主导

你主导：

- 每条主题想训练哪种思维动作
- level 的命名和边界
- 升级门槛

### 2. 系统来主导

系统主导：

- 当前属于哪一层 `adaptationLevel`
- 最近几次是 `too_easy / fit / too_hard`
- 下一次 seed 选更轻还是更高半级

### 3. AI 来主导

AI 主导：

- 这一轮怎么开口
- 怎么接住孩子的话
- 怎么追问
- 怎么给两个方向

## 主题 1：观察规律（pattern）

### 核心目标

让孩子：

- 先看到什么在重复或变化
- 再说清这条规律
- 最后能在换一种表示方式时继续认出来

### L1：进入与看见

目标：

- 看见明显重复
- 说出下一步大概是什么

当前 seed：

- `pattern-scarf-1`
- `pattern-steps-1`

判断“可以进 L2”时，看这些信号：

- 最近 2-3 次里，不只是猜下一项
- 开始能说“哪里一样 / 哪里轮流”
- 支架依赖不高

### L2：说清与比较

目标：

- 不只是看见规律，还能开始说清
- 能在两个解释里比较哪个更像

当前 seed：

- `pattern-tower-1`
- `pattern-clap-1`

判断“可以进 L3”时，看这些信号：

- 能说“每次怎么变”
- 能比较两个可能规则
- 至少一次能挡住一个不像的答案

### L3：换条件与迁移

目标：

- 换一种表示方式、方向、布局，孩子还能继续跟上

当前 seed：

- `pattern-window-1`
- `pattern-turn-1`

稳定信号：

- 换一种呈现后仍能抓住结构
- 一换条件不会立刻丢失方向

## 主题 2：为什么与解释（why）

### 核心目标

让孩子：

- 愿意大胆猜一个原因
- 再把理由说得更完整一点
- 最后开始考虑“如果换个情况，原因会不会变”

### L1：进入与看见

目标：

- 肯开口猜原因
- 愿意说一句“我觉得是因为……”

当前 seed：

- `why-shadow-1`
- `why-ice-1`

进 L2 的信号：

- 不只给单词答案
- 开始补上因果词
- 不需要太多 choices 才能继续说

### L2：说清与比较

目标：

- 能比较两个可能原因
- 能开始分“看到的现象”和“我猜的原因”

当前 seed：

- `why-plant-1`
- `why-rain-1`

进 L3 的信号：

- 能在两个原因里偏向一个并说理由
- 会提到“如果换一下，也许会不同”

### L3：换条件与迁移

目标：

- 不只想物理现象，也能处理人和情绪里的多种可能
- 能从“猜原因”走到“如果这样我会怎么做”

当前 seed：

- `why-friend-1`
- `why-socks-1`

稳定信号：

- 能提出两个以上可能
- 能把解释推进到办法或回应

## 主题 3：公平与选择（fairness）

### 核心目标

让孩子：

- 先觉得“这件事好像不太公平”
- 再比较不同规则为什么更合适
- 最后能接受“情境一变，规则也可能要变”

### L1：进入与看见

目标：

- 能开始区分“一样多”和“公平”不总是一回事

当前 seed：

- `fairness-cookies-1`
- `fairness-lineup-1`

进 L2 的信号：

- 能说出自己选规则的一个理由
- 开始看到“照顾谁”这件事

### L2：说清与比较

目标：

- 能比较两条规则各自照顾了什么
- 能提出一条更合理的新规则

当前 seed：

- `fairness-cleanup-1`
- `fairness-game-1`

进 L3 的信号：

- 不是只说公平/不公平
- 而是开始改规则并说明理由

### L3：换条件与迁移

目标：

- 一旦人变了、情境变了，能重新调整规则

当前 seed：

- `fairness-team-1`
- `fairness-story-1`

稳定信号：

- 能接受规则不是固定不变的
- 会根据新情况重新思考“谁先 / 谁定 / 谁更需要被照顾”

## 主题 4：假设与预测（what-if）

### 核心目标

让孩子：

- 先愿意进入一个假设世界
- 再说第一步会发生什么
- 最后开始考虑规则、系统或后续变化

### L1：进入与看见

目标：

- 能顺着一个具体假设说出第一件变化

当前 seed：

- `whatif-redlight-1`
- `whatif-chair-1`

进 L2 的信号：

- 不只说“会乱”
- 而是能指出先会发生哪件具体事

### L2：说清与比较

目标：

- 能想象更丰富的变化
- 能比较两个可能后果

当前 seed：

- `whatif-two-moons-1`
- `whatif-book-1`

进 L3 的信号：

- 能从“变化了什么”推进到“那会影响什么”

### L3：换条件与迁移

目标：

- 能把假设推进到规则层或系统层
- 不只是想一个画面，而是开始想结构会怎样变

当前 seed：

- `whatif-rule-1`
- `whatif-rainbow-road-1`

稳定信号：

- 能从“如果”推进到“那规则要不要改 / 提醒方式要不要改”

## 当前运行规则

非数学主题当前已经具备轻量适应：

- 每次会话结束时
  - 记录 `difficultySignal`
  - 记录 `supportLevel`
  - 记录 `adaptationLevel`
  - 记录 `nextSuggestedLevel`

- 下一次开始时
  - 读取最近若干条同主题 session log
  - 如果最近 `2/3` 次偏轻松，则升一级
  - 如果最近 `2/3` 次偏吃力，则降一级
  - 否则保持当前级

对应代码：

- `lib/daily/theme-adaptation.ts`

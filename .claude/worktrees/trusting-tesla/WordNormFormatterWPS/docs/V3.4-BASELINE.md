# v3.4 Baseline

Date: 2026-03-24

This tag freezes the current formatter behavior before the next architecture shift.

## Scope

- Stabilize heading recognition and outline levels for one-click formatting.
- Preserve original automatic numbering on real section headings.
- Strip numbering from `docTitle` / main title paragraphs.
- Make TOC insertion and section breaks conservative on shorter documents.
- Move header/footer pagination updates to the end of the pipeline.
- Keep source files and `wps-addon-build` artifacts in sync.

## Known Direction After This Baseline

The next phase moves from:

- AI classifies paragraphs
- code hardcodes formatting decisions

to:

- AI understands document structure and emits a formatting plan
- code executes that plan deterministically through WPS APIs

## Intent

Use `v3.4` as the rollback point before introducing the AI planning layer.
